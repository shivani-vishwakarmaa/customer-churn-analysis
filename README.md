# Customer Churn Analysis & Retention Insights

Customer churn is a major problem for many businesses, especially in subscription-based industries like telecom, SaaS, and streaming services. Losing customers directly affects revenue, which is why companies invest heavily in understanding **why customers leave and how to retain them**.

In this project, I performed an **end-to-end Customer Churn Analysis** using telecom customer data. The goal was to identify churn patterns, understand the factors influencing churn, and present insights through an interactive dashboard.

---

# Project Objective

The main objectives of this project were:

* Identify **which customers are more likely to churn**
* Understand **key factors contributing to churn**
* Analyze **customer behavior across different plans and services**
* Create a **business-friendly dashboard** for decision-making
* Provide **data-driven recommendations** to reduce churn

# Dataset

The dataset used in this project contains telecom customer information such as:

* Customer demographics
* Contract type
* Internet service
* Payment method
* Monthly and total charges
* Customer tenure
* Churn status

The dataset helps simulate a real-world scenario where companies analyze customer behavior to reduce churn.

# Project Workflow

The project followed a typical data analytics workflow:

### 1. Data Cleaning

* Checked missing values
* Converted incorrect data types
* Removed inconsistencies in the dataset

### 2. Exploratory Data Analysis (EDA)

* Analyzed churn distribution
* Compared churn across contract types
* Studied the relationship between monthly charges and churn
* Investigated customer tenure patterns

### 3. SQL Analysis

* Wrote queries to analyze churn trends
* Calculated churn counts across different customer segments

### 4. Dashboard Development

Built an interactive **Power BI dashboard** to visualize key churn metrics and patterns.

---

# Dashboard Overview

The Power BI dashboard highlights important metrics such as:

* Total Customers
* Total Churned Customers
* Churn Rate
* Churn by Contract Type
* Churn by Internet Service
* Churn by Payment Method
* Monthly Charges vs Churn
* Customer Tenure Analysis

The dashboard is designed to help business stakeholders **quickly understand churn behavior and make informed decisions**.

---

# Key Insights

Some interesting insights from the analysis include:

* Customers with **month-to-month contracts show the highest churn rate**
* Customers with **higher monthly charges tend to churn more**
* **New customers (low tenure)** are more likely to leave
* Certain **payment methods show higher churn patterns**

These insights can help businesses focus their retention strategies more effectively.

---

# Business Recommendations

Based on the analysis, the following strategies could help reduce churn:

* Encourage customers to move from **month-to-month contracts to long-term plans**
* Provide **loyalty incentives for long-tenure customers**
* Improve customer experience for **high-cost plans**
* Offer **personalized retention offers** for at-risk customers
